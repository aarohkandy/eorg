Before You Connect

We need one thing before you can sign in: a Google App Password for the Gmail account you want to sync.

  • Open App Passwords for the Google account you want to connect.
  • If Google says App Passwords is unavailable, turn on 2-Step Verification first.
  • Create a new password for Mailita and copy the 16-character code.
  • Come back here and paste that code on the next step.
